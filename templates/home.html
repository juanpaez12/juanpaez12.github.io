Estructura de Datos

¿Qué es una estructura de datos?

Fig1: https://estructurayorganizaciondedatos.blogspot.com/2013/09/unidad-i-fundamentos-de-estructura-de.html

Una estructura de datos es una forma de organizar un conjunto de datos elementales con el objetivo de facilitar su manipulación. Una estructura de datos define la organización de estos y un conjunto de operaciones que se pueden realizar sobre ellos.
Ventajas:
- Permite modificar “Globalmente” las variables sin necesidad de recorrer el código buscando cada uno de ellos.
- Define variables y evita que cambien entre rutinas.
- Separa desde el inicio del programa el espacio en memoria.
- Se trabaja más fácil el paso de datos entre aplicaciones.

Desventajas:
- Puede causar confusión al intentar unir o pasar información entre programas si no se tiene en cuenta la longitud y tipo de datos.
- Si la memoria es poca, puede separar el espacio que nunca se utiliza y podría hacer que el equipo no responda.
