                public class ListQueue implements QueueInterface {


                    private int size;
                    private ListNode top;
                    private ListNode bot;
                    
                    
                    public ListQueue() {
                        clear();
                    }

Este método es utilizado para limpiar la lista.

            

            
        

            
                @Override
                public void clear() {
                    size = 0;
                    top = null;
                    bot = null;
                }

Inicialización de las variables

            
                @Override
                public boolean isEmpty() {
                    return top == null;
                }

Verifica si la cola está vacía

            
                @Override
                public Object extract() {
                    try{
                        if(!isEmpty()){
                            Object a = top.getObject();
                            top = top.next;
                            size--;
                            return a;
                        }else{
                            return null;
                        }    
                    }catch(Exception e){
                        return null;
                    }
            
                }

Extrae el primer elemento añadido y lo elimina

            
                @Override
                public boolean insert(Object object) {
                    try{
                        ListNode nodoInsertado = new ListNode(object);
                        if(!isEmpty()){
                            bot.next = nodoInsertado;
                        }else{
                            top = nodoInsertado;
                        }
                        bot = nodoInsertado;
                        size++;
                        return true;
                    }catch(Exception e){
                        return false;
                    }
                }

Inserta un nuevo nodo a la cola

            
                @Override
                public int size() {
                    return size;
                }

Permite retornar el tamaño de la cola

            
                @Override
                public boolean search(Object object) {
                    
                    try{
                        if(!isEmpty()){
                            ListNode a = top;
                            while(a != null){
                                if(a.getObject().equals(object)){
                                    return true;
                                }
                                a = a.next;
                            }
                            return false;
                        }else{
                            return false;
                        }
                    }catch(Exception e){
                        return false;
                    }
                }

Permite buscar un elemento especifico de la cola.

            
                @Override
                public void sort() {
                    try{
                        Object[] arreglo = new Object[size];
                        ListNode a = top;
                        for (int i = 0; i < size; i++) {
                            arreglo[i] = a.getObject();
                            a = a.next;
                        }
                        java.util.Arrays.sort(arreglo);
                        top = null;
                        for (int i = 0; i < arreglo.length; i++) {
                            insert(arreglo[i]);
                        }
                        
                    }catch(Exception e){
                        
                    }
                }

Este método permite ordenar los elementos de la cola por medio de un arreglo

            
                @Override
                public void reverse() {
                    if(!isEmpty()){
                        Object[] array = new Object[size];
                        ListNode a = top;
                        for (int i = 0; i < size; i++) {
                            array[i] = a.getObject();
                            a = a.next;
                        }
                        top = null;
                        for (int i = array.length-1; i > -1; i--) {
                            insert(array[i]);
                        }
                        
                    }
                }

Permite mostrar los elementos de la cola de manera inversa

            
                @Override
                public String toString() {
                    if(!isEmpty()){
                        return top.toString();
                    }else{
                        return null;
                    }
                    
                }
